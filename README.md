# BatteryLifeLLM
## Dependencies
You will need at least 8 GTX3090 to run the 7B version for BatteryLifeLLM.

run the following command to install the dependecies:

(This is generated by pipreqs tool)
```
pip install -r requirements.txt
```

## Dataset preprocessing
Gnerally, we follow [BatteryML](https://github.com/microsoft/BatteryML) to preprocess the datasets otherwise specified. The details of the preprocessing in this work is describe  [here](Split_datasets/README.md).

## Run the code
```
sh scripts/exp.sh

# if you want to run the code in the back end
screen -S batteryLifeLLM
nohup sh scripts/exp.sh &> ./outs/out&
CTRL A+D to detach from the sceen
```
